Добавляем бокс в Vagrant

`vagrant box add ubuntu-18.04.1.box --name alex-ubuntu1804test`

Инициализируем его

`vagrant init alex-ubuntu1804test`

Поднимаем виртуальную машину

`vagrant up`

Для подключения к машине используем

`vagrant ssh`
